Auroria

Auroria is a 2D physics based puzzle game, which I made together with five students for a project.

Project Final Approach

Auroria is the result of project Final Approach, a project in the first year of Creative Media & Game Technologies. The assignment was to make a physics based puzzle game in the GXPEngine, a 2D engine made by school. As I was the only engineer in our team of six people, I had to do all the programming. This was quite difficult for me, but luckily I could apply the knowledge I learned in a previous physics programming course about collisions and gravity.
